# Skin Cancer Detection

Senior Design Project website and milestone archive for a mobile app focused on real-time skin cancer risk screening using machine learning.

## Overview

The Skin Cancer Detection project is a mobile application concept designed to help users capture or upload images of suspicious skin lesions and receive a preliminary AI-assisted risk assessment. The system is intended to support early awareness, encourage timely medical follow-up, and improve access to screening support for users who may not have immediate access to dermatology care.

This repository currently hosts the project website and supporting milestone documents for the senior design course.

## Project Goals

- Build a mobile-friendly skin lesion screening application
- Provide real-time, AI-assisted preliminary risk feedback
- Keep user data privacy-focused through on-device inference
- Encourage professional consultation when higher-risk results are detected
- Present project milestones, evaluations, and deliverables in a clear public website

## Key Features

- Image capture and image upload workflow
- Real-time lesion analysis using machine learning
- On-device inference using TensorFlow Lite
- Educational guidance and medical disclaimer support
- Cross-platform mobile development approach using Flutter
